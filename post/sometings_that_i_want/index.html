나의 현재 수준과 앞으로 하고 싶은 내용..

Fri, Dec 11, 2020

현재 상태

데이터 엔지니어가 되고 싶다.

지금은 거의 모른다고 말할 수 있을거 같다.

거의 모르는 상태에 대해서 작성해보자면

Git

  • 깃, 깃허브 개인적으로 혼자 블로그하면서 작성해봐서 아주 큰 개념, commit, push, clone 만 쓰는 정도이다.
  • 특별한 문제를 만나본적이 없다. 교과서에서 나오는 충돌문제도 아직 한번도 경험이 없는 상태이다.
  • 혼자서 사용했기 때문에 하나의 branch를 기준으로 개인 레포지토리에 올리고 받아보는 정도이다.
  • 연습하고 사용해볼겸 Hugo 를 이용한 블로깅을 시작하였다.

블로깅

  • Git과 블로깅을 연습해볼겸 Hugo를 이용한 블로그를 시작하였다.
  • Hugo의 튜토리얼만 따라해서 구축했을 뿐, 추가적인 작업은 아예하지 않았다.
  • 추가적인 작업에 어떤 개념이 있는지도 모르는 상태이다.

통계

  • 스터디파이라는 스터디 사이트를 통해 통계학을 들었다.
  • 스터디에 모두 참가하고, 스터디 내용을 노트필기하면서 기록하였다.
  • 그러나 크게 머리에 남지 못하였다. 문제는 아직 모르곘다. 계속 사용하지 못해서 그런가
  • 일반적인 대학교 1학년 기초통계학 수업을 들으면 B 정도일거 같다.

머신러닝,딥러닝

  • T 아카데미, 산기협 강의 등을 통해서 머신러닝 및 딥러닝 강의에 대해서 많이 들었다.
  • 짧은 코스로 된 강의를 많이 들었다. 하루에 8~9시간 4일정도.. Youtube 를 통해서도 많이 들은 듯하다.
  • 간단하게 사용하라면 할수 있을 것이다. 그러나, 내부의 동작 방식 하나도 모르채로 쓸거 같다.
  • 머신러닝, 딥러닝을 아는 사람들 중에 최하위 이해 수준이라고 생각한다.

컴퓨터과학 (자료구조, 알고리즘)

  • 학과 수업을 한번들은 정도이다.
  • 엔지니어링에 녹여내지 못하고 있다.
  • 프로그로머스 문제들을 레벨 3까지 해봤다.

엔지니어링 (백엔드, SQL, 웹 서버, DB, Python, Shell Script, 객체지향프로그래밍 )

  • 접해봤다. 위에 기술들을 접해만 봤다. 정도가 맞는거 같다.
  • 그 기술에 대한 개념을 한, 두줄 정도로 알고 있고, 그 기술이 쓰는 기능 제일 간단하게 쓸 수 있는 정도이다.
  • 다양한 기술들이 연계되는데 내가 아는 정도는, 참… 단순하게 안다고 생각한다.
  • 해당 분야에 대해서 아는 사람들 중에 최하위 이해 수준이라고 생각한다.

바뀌고 싶은점

이 글을 쓰게 된건 계획을 짜고 써볼려고 했는데 그 계획대로 안지켜져서 우선 무엇이라도 블로그에 올려보자는 생각에서 시작하였다. 어제 안쓰고 그제 안썻는데 오늘도 안쓰다보면 안쓰는게 습관화되지는 않을까 걱정이 되어서 시작하였다. 무엇이라도 올리다보면 스스로가 바뀌지 않을까…

쓰다보면서 든 생각은 한번씩 이런식으로 쓰다보니까 이 내용을 잘 쓴거같다는 생각이 들었다.

  • 하나는 나의 현재 가치에 대해서 객관적으로 살펴볼 수가 있고
  • 두번째는 머리속에 스트레스처럼 있는 막연한 두려움에 대해서 약간은 정리가 되는거 같다. 막연한 두려움이 너무 힘들어서 노트에 써보면서 정리하려고 했는데.. 그것도 그 한순간 일뿐 노트를 다시 보지 않는 경우가 태반이 되는거 같았다. 블로그는 낫지않을까.. 아직 모르겠다.

앞으로 어떻게 나아갈것인가

Git

  • Git 한번 사용해보도록 하겠다.

블로깅

  • 한 달 2, 4주 토요일에 블로그를 정비하도록 하겠다.

통계 컴퓨터과학 (자료구조, 알고리즘)

  • 이 두개만큼은 경험보다 이론인거 같다. 다시 한번 정리해보도록 하곘다.

머신러닝,딥러닝 엔지니어링 (백엔드, SQL, 웹 서버, DB, Python, Shell Script, 객체지향프로그래밍 )

  • 위 두개는 계속 공부하는 건 큰 의미가 없을 것이라고 생각한다.
  • 토이프로젝트를 직접하면서 자료를 정리해보도록 하겠다.
  • 하지만 엔지니어링 파트를 할 때는 단순히 사용에 방점을 두지 않고, 내부 프로세스에 대해서 정확히 알도록 보면서 하도록 하겠다.
  • 우선순위 엔지니어링 > 머신러닝,딥러닝

무언갈 자꾸 쓰다보면 언젠가는 내가 다듬지 않을까..

꾸준히 작성하다보면 스스로 각성하는 계기도 오지 않을까

어제도 그제도 어떤 글도 작성하지 못한거 같아 글을 한번 올려본다
